# 🚁 FPV Drone Telemetry Dashboard

[demo gif here]

Real-time UAV telemetry dashboard for Betaflight flight controllers.
Built with Python, WebSockets and React.

## Features
- Live battery voltage, attitude and motor output
- 3D drone visualizer with real-time rotation
- Auto-detection of flight controller COM port
- Automatic reconnection with port selector

## Tech Stack
- Python (pyserial, websockets, MSP protocol)
- React (via CDN, no build step)
- Three.js (3D visualization)
- Chart.js (live graphs)

## Hardware Used
- BETAFPV Meteor 75 Pro
- Betaflight 4.x
